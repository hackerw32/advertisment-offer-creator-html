## ğŸ’¡ Tips & Best Practices

### Design Tips

1. **Keep it Simple**
   - Don't overcrowd with too many offers
   - Use white space effectively
   - Focus on 1-2 key messages

2. **Color Psychology**
   - Gold/Yellow: Premium, luxury
   - Blue: Trust, professional
   - Red/Orange: Urgency, excitement
   - Green: Natural, eco-friendly

3. **Typography**
   - Keep titles short and impactful
   - Use descriptions to provide details
   - Make prices prominent and easy to read

4. **Images**
   - Use high-quality logos (300+ DPI)
   - Background images should not overwhelm text
   - Adjust opacity for better text readability

### Export Tips

1. **For Printing:**
   - Use PDF A4 export
   - Check colors on screen match printer
   - Test print one copy before bulk printing

2. **For Social Media:**
   - Use Instagram export (1080x1080)
   - Test on mobile devices
   - Ensure text is readable at small sizes

3. **For Booklets:**
   - Plan content across all pages first
   - Maintain consistent design
   - Consider page order when folding

## ğŸ› Troubleshooting

### Common Issues

**Template not saving:**
- Check browser localStorage is enabled
- Clear browser cache and try again
- Ensure cookies/storage is not blocked

**Export not working:**
- Disable browser popup blocker
- Try different browser
- Check file download permissions

**Images not uploading:**
- Check file size (keep under 5MB)
- Use standard formats (JPG, PNG)
- Check file permissions

**Colors look different when printed:**
- Use CMYK-friendly colors
- Test print before bulk printing
- Adjust monitor color calibration
